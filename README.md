# agentic_github_workflows
Exploring the Github agentic automation for an accelerated SDLC

## Démarrage rapide (Angular 16+)

### Prérequis
- Node.js >= 16.x
- npm >= 8.x

### Installation

1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer le serveur de développement :
   ```bash
   npm start
   ```
3. Accéder à l’application :
   Ouvrez [http://localhost:4200](http://localhost:4200) dans votre navigateur.

### Commandes utiles
- `npm run build` : Générer le build de production
- `npm test` : Lancer les tests unitaires
- `ng lint` : Vérifier la qualité du code

Pour plus d’informations, consultez la documentation officielle Angular : https://angular.io/docs
